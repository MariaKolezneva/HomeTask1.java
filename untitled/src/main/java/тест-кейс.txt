Тест-кейсы:
1. Ввести два целых числа (36788; 25) - Выводится результат: суммы, вычитания, умножения, деления.
2. Вести два дробных числа (2.5456; 42.51) - Выводится результат: суммы, вычитания, умножения, деления.
3. Ввести одно дробное, одно целое число (54; 43.12) -Выводится результат: суммы, вычитания, умножения, деления.
4. Ввести отрицательное, положительное число (-34; 436)  - Выводится результат: суммы, вычитания, умножения, деления.
5. Ввести "exit" - происходит выход.
6. Ввести "-0" - определяется как "0".
Негативные:
1. Вевсти невалидные значения в первый аргумент (ав&%с";45) (буквы, символы) - Отображается ошибка;
2. Вевсти невалидные значения во второй аргумент (57; ав@с) (буквы, символы) - Отображается ошибка;
3. Вевсти невалидные значения в оба аргумента (abc;a#jg) (буквы, символы) - Отображается ошибка;
4. Поделить на ноль - Отображается ошибка;
5. Ввести null в первый аргумент - Отображается ошибка;
6. Ввести null во второй аргумент - Отображается ошибка;
7. Ввести null в оба аргумента- Отображается ошибка.
8. Ввести ноль в два аргумента - Отображается ошибка при делении.
