Тест-кейсы:
1. Ввести два целых числа (3; 25) - Выводится результат: суммы, вычитания, умножения, деления.
2. Вести два дробных числа (2.54; 42.1) - Выводится результат: суммы, вычитания, умножения, деления.
3. Ввести одно дробное, одно целое число (54; 43.12) -Выводится результат: суммы, вычитания, умножения, деления.
4. Ввести отрицательное, положительное число (-34; 43)  - Выводится результат: суммы, вычитания, умножения, деления.
Негативные:
1. Вевсти невалидные значения в первый аргумент (ав&%с";45) (буквы, символы) - Отображается ошибка;
2. Вевсти невалидные значения во второй аргумент (57; ав@с) (буквы, символы) - Отображается ошибка;
3. Вевсти невалидные значения в оба аргумента (abc;a#jg) (буквы, символы) - Отображается ошибка;
4. Ввести "-0" в первый аргумент - Отображается ошибка;
5. Ввести "-0" во второй аргумент - Отображается ошибка;
6. Ввести "-0" в оба аргумента - Отображается ошибка;
7. Поделить на ноль - Отображается ошибка;
8. Ввести null в первый аргумент - Отображается ошибка;
9. Ввести null во второй аргумент - Отображается ошибка;
10. Ввести null в оба аргумента- Отображается ошибка.
